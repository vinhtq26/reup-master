# gui

Thư mục này chứa các file giao diện người dùng (User Interface - UI) cho dự án Video Downloader:

- `video_downloader.py`: Ứng dụng giao diện đồ họa (GUI) sử dụng customtkinter, cho phép tải video, quản lý kênh, xem thống kê, ...
- `video_downloader_cli.py`: Ứng dụng giao diện dòng lệnh (CLI) đơn giản, không cần GUI, phù hợp cho môi trường server hoặc không có giao diện đồ họa.

Các file trong thư mục này giúp người dùng tương tác trực quan với các chức năng của hệ thống.
