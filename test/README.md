# test

Thư mục này chứa các file kiểm thử (test scripts) cho dự án Video Downloader:

- `test_first_scan.py`, `test_instant_stop.py`, `test_quality.py`, `test_stop_monitoring.py`, `test_user_video.py`, `test_video_processing.py`: Các script kiểm thử chức năng tải video, kiểm tra logic, kiểm tra dừng monitoring, kiểm tra chất lượng, ...

Bạn có thể chạy các file này để kiểm tra tính ổn định và đúng đắn của các chức năng chính trong hệ thống.
