# scripts

Thư mục này chứa các script hỗ trợ chạy nhanh, ví dụ, hoặc hướng dẫn sử dụng cho dự án Video Downloader:

- `quick_start.py`: Hướng dẫn sử dụng nhanh, kiểm tra môi trường và các bước chạy cơ bản.
- `quick_download.py`: Script tải video nhanh từ dòng lệnh, không cần giao diện.

Các file trong thư mục này giúp bạn kiểm tra, chạy thử hoặc hướng dẫn sử dụng dự án một cách nhanh chóng.
