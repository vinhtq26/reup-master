# config

Thư mục này chứa các file cấu hình cho dự án Video Downloader:

- `config.py`: Các biến cấu hình chính như số kênh tối đa, nền tảng hỗ trợ, thời gian kiểm tra, ...
- `user_settings.json`: Lưu các tuỳ chọn cá nhân của người dùng (thư mục lưu, tuỳ chọn upload, ...).
- `credential.json`, `credentials.json`, `token.json`: Thông tin xác thực cho các dịch vụ bên ngoài (ví dụ: Google Drive).
- `README.md`: Hướng dẫn cấu hình và giải thích các file trong thư mục này.

Bạn chỉ cần chỉnh sửa các file trong thư mục này khi muốn thay đổi cấu hình hệ thống hoặc thông tin xác thực.
